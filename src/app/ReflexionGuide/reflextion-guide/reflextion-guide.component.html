enjeux

Dans la feuille "Enjeux" vous êtes censés de rapporter vos enjeux. En identifiant les Forces et les faiblesses vous êtes entrain d'identifier vos enjeux internes. Cependant, en identifiant les Opportunités et les Menaces vous êtes entrain d'identifier vos enjeux externes. Vous trouvez des exemples de différentes catégories pour les classifier. De plus, il y a un exemple de formulation d'un enjeu dans le début de chaque tableau. Un tableau de synthèse se trouve à la fin pour consolider tous les enjeux et un graphe pour visualiser leur répartition dans les catégories. Vous trouvez également d'autre synthèse dans la feuille "Synthèse".

Cette multitude de questions vous aideront rafraîchir votre mémoire afin d'étudier tous les éléments qui peuvent vous engendrer des enjeux .

Périmètre_Politique_Processus

Dans la feuile "Périmètre_Politique_Processus" vous êtes censé de: transcrire votre politique dans l'espace qui lui est dédié, définir le périmètre de votre système management dans son espace et identifier vos processus. Vous devez définir le pilote de chaque processus ainsi que ses indicateurs de performances. Le tableau dynamique à côté du tableau des processus sert à la récapitulation des indicateurs de chaque process qui va vous aider dans les synthèses définies dans la feuille "Synthèse".

Parties intéressées

Dans la feuille "Parties intéressées" vous êtes censés d'identifier vos parties intéressées. Vous trouvez un exemples de différentes catégories pour les classer. A la suite, il y aura la partie d'évaluation des parties intéressées afin d'extraire ceux qui sont pertinentes à l'aide de la matrice Pouvoir/ Influence ou la matrice Intérêt/Influance dont vous trouvez l'explication dans le bouton "Loupe". Ainsi les parties intéressées pertinentes seront synthétiser dans le tableau qui suit avec leurs poids. Vous pouvez aussi choisir d'identifier les exigences de ces parties dans l'outil ou ailleurs. Vous trouvez également d'autre synthèse dans la feuille "Synthèse".
